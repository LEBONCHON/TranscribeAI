# **App Name**: LinguaFlow

## Core Features:

- Live Speech Recording to Text: Record spoken audio and convert it into transcribed text in real-time using speech-to-text AI.
- Text-to-Voice Generation: Generate spoken audio from text input, allowing users to hear the translated text.
- Language Selection: Allow users to select the input and output languages for both voice-to-text and text-to-voice functionalities.
- Text Copy Functionality: Enable users to copy the transcribed or translated text for easy sharing and use elsewhere.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), embodying trust and clear communication.
- Background color: Light blue (#E0F7FA), which is soft and unobtrusive, suitable for extensive reading or focus-intensive use.
- Accent color: Cyan (#00FFFF) is employed for interactive elements and call-to-action buttons.
- Font pairing: 'Inter' sans-serif for both headings and body text to maintain a modern and neutral look.
- Use clean, line-based icons for language selection, recording, and text manipulation functionalities.
- A clean, intuitive layout that clearly distinguishes between voice input and text output sections.
- Subtle animations to indicate voice input and processing to the user, like pulsating mic button while recording.
